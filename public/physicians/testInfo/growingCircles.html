This is the Growing Circles Test. The aim of of it is to detect if the user has any blind spots in their vision, as well as the size of said spots. While the user is focusing on the black dot in the center of the grid, red circles will start to appear on the canvas. The user must click a button when they see the circle. If the user does not react when the circle appears on the screen, the circle will start to grow until the user sees it. This will detect where the user has a blindspot and how big of a blindspot it is.

The test starts off with a blank grid. The user is instructed to cover the eye they are not testing before they start. The eye being tested must be focusing on the black dot in the center the entire time.

During the test, red dots will randomly be populated around the grid, one by one. All dots start off at the same size, and the locations are randomized. The user should click a button when they see the circle. After clicking the button, the current circle will disappear and a new one will be generated.

When a circle is generated and the user does not click the 'seen' button, the circle will grow. It will continue to grow until the user clicks the button to indicate that they now see it.

The results of the Growing Circles Test is one canvas that displays all the circles the user missed during the test. The circles missed are determined based on growth. If the user detected the circles as soon as they appeared, they will not be shown in the test results. Only circles intially missed and forced to grown will be marked. In this example, the user missed three circles. All three circles are on the top left of the user's centeral vision. This could mean the user has a blind spot in that area, and should be checked by a professional.
