This is the Smiley Test. The aim of of it is to detect if the user has any blind spots in their vision, as well as to detect any vision abnormalities. While the user is focusing on the black dot in the center of the grid, different facess will start to appear on the canvas. The user must click "S" if a "smiley face" appears and "F" if a "frowny face" appears. If the user does not click the correct key response the location of that error is flagged on the Canvas.

The test starts off with a blank grid. The user is instructed to cover the eye they are not testing before they start. The eye being tested must be focusing on the black dot in the center the entire time.

During the test, different faces will randomly be populated around the grid, one by one. The locations of all the symbols are randomized. The user should click the corresponding button to each face.

The results of the Smiley Test is one canvas that displays all the symbols the user missed during the test.
